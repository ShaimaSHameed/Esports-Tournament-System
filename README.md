# Esports Tournament Management System
**Database Systems Course · Spring 2026**

Relational database system for managing esports tournaments, covering teams, players, matches, sponsors, and venues. Includes a Java GUI connected live to Oracle DB supporting insert, update, delete, and search across 15 tables.
